# my.crm

Лёгкая CRM-система для отдела продаж техники (CAT / SEM).  
Работает полностью в браузере — без сервера, без базы данных.

## Возможности

- 👥 Клиенты — добавление, редактирование, фильтрация, поиск
- 🔥 Воронка продаж с вероятностью закрытия сделки
- 📋 Задачи — звонки, встречи, заметки с напоминаниями
- ⚠️ Вата-клиенты — отдельный список потерянных лидов
- ✅ Проданные — история закрытых сделок
- 🧮 Калькулятор GP — расчёт маржи, ЗИП, DDP (НДС 12% + Пошлина 5.4% + НСП 1%)
- 🎮 Персонаж — геймификация продаж
- 💬 Командный чат
- 📊 Экспорт / Импорт JSON и Excel
- 🌓 Две темы: CAT (тёмная) и SEM (светлая)
- 📱 Полная адаптивность — работает на телефоне, планшете и ПК

## Запуск

### Вариант 1 — GitHub Pages (рекомендуется)

1. Загрузите `index.html` в репозиторий
2. Перейдите в `Settings → Pages`
3. Выберите ветку `main`, папку `/ (root)`
4. Сохраните — сайт будет доступен по адресу `https://ВАШ_НИК.github.io/ИМЯ_РЕПО/`

### Вариант 2 — локально

Просто откройте `index.html` в браузере. Данные сохраняются в `localStorage`.

## Данные

Все данные хранятся в `localStorage` браузера. Используйте кнопку **Экспорт JSON** для резервной копии.
